This documentation is still a work in progress. If you have any suggestions or
feedback, please feel free to open an issue or pull request on the [GitHub repository](https://github.com/miona-cc/docs/tree/main).

## What is Miona?
Miona is a Discord bot that allows you to create and share your own profile
pictures. It is powered by [Discord.js](https://discord.js.org/) and uses the
[Discord API](https://discord.com/developers/docs/intro) to interact with the
server.
